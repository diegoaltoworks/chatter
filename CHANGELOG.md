# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Initial release of Chatter framework
- RAG-powered chatbot with OpenAI embeddings
- Turso vector database integration
- Customizable chat widgets (ChatBot, Chat, ChatButton)
- Authentication support (JWT, Clerk, custom providers)
- Mobile-optimized responsive design
- Rate limiting and CORS middleware
- Security guardrails for input validation
- TypeScript support with full type definitions
- Streaming response support
- Knowledge base management
- Session management
- Prompt template system

### Documentation
- Comprehensive README with quick start guide
- Contributing guidelines
- Code of Conduct
- Issue and PR templates
- API documentation
- Example implementations
